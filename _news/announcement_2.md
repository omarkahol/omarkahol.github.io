I was selected to be part of the <a href="https://traces-project.eu/">TRACEs</a> project as PhD student. This position will be founded by <a href="https://marie-sklodowska-curie-actions.ec.europa.eu/">MSCA</a> scholarship and consists of a double degree between <a href="https://www.polytechnique.edu/en">Ecole Polytechnique</a> and <a href="https://www.polimi.it"> Politecnico di Milano</a>. 

***

The tile of the PhD is "Robust assessment of the model error for ice accretion models through Bayesian-based methods and experimental data".


#### Goals
<ul>
    <li>introduce a robust procedure for assessing the model error for any model used during the simulation</li>
    <li>calibrate model parameters considering systematically the model error</li>
    <li>combine several models’ predictions to improve the overall prediction capabilities</li>
</ul>
